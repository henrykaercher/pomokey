# PomoKey

Simple Pomodoro timer built with raylib.

## Compiling
Simply use the Makefile:
```bash
make
```
Or if you want to compile and run:
```bash
make run
```

## Notes
- Window resizing is disabled due to incorrect mouse coordinates
  under XWayland + Hyprland.
- This software is unfinished.
- No Windows versions on my plans for this project.
